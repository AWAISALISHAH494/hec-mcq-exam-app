# HEC National Skills Competency Test & Academy Curriculum

This repository contains two main resources:
1. **HEC National Skills Competency Test Web App**: An interactive, 100-question mock exam designed specifically to test the competencies outlined by the Higher Education Commission (HEC) of Pakistan.
2. **Full-Stack & DevOps Academy Curriculum**: A comprehensive set of study plans and roadmaps for an intensive tech academy.

---

## 1. HEC Mock Exam Web App
The `hec_mcq_test.html` file is a completely self-contained, interactive web application. It requires no installation or server to run.

### Features
* **100 Comprehensive Questions** covering:
  * Computer Networks & Cloud Computing (10%)
  * Programming (C++/Java/Python) (10%)
  * Data Structures & Algorithms (10%)
  * Operating Systems (5%)
  * Software Engineering (10%)
  * Web Development (10%)
  * AI / Machine Learning & Data Analytics (10%)
  * Cyber Security (5%)
  * Databases (10%)
  * Problem Solving & Analytical Skills (20%)
* **Instant Feedback**: Highlights correct answers in green and incorrect selections in red upon submission.
* **Score Calculation**: Automatically tallies your score out of 100 at the top of the page.

### How to Use
Simply double-click the `hec_mcq_test.html` file to open it in any modern web browser (Chrome, Firefox, Edge, Safari). Select your answers and click **Submit Test & Get Score** at the bottom!

## 2. Academy Curriculum & Study Plans
Included in this repository are meticulously crafted, day-by-day (8-10 week) study plans for software engineering and infrastructure domains:

1. **Python Programming (`python_course_plan.md`)**: From variables to OOP to APIs and file handling.
2. **Django & DRF (`django_drf_course_plan.md`)**: From MVC web basics to advanced JWT authenticated REST APIs.
3. **DevOps Engineering (`devops_course_plan.md`)**: Covering Linux, Git, Docker, CI/CD, AWS, Terraform, and Kubernetes.
4. **Linux for DevOps Guide (`linux_for_devops_guide.md`)**: A massive hands-on tutorial and practice lab for server management.
5. **DevOps Roadmap (`devops_roadmap.md`)**: A zero-to-hero guide on the modern DevOps tech stack.
